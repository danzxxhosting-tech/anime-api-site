Quotes Anime

Endpoint untuk mendapatkan kutipan anime.

GET
/api/quotes

Parameter:

  • limit: Jumlah hasil (default: 1)
  • anime: Filter berdasarkan judul anime
{
  "status": "success",
  "data": [
    {
      "quote": "Percaya pada dirimu sendiri!",
      "character": "Naruto Uzumaki",
      "anime": "Naruto Shippuden"
    }
  ]
}

Gambar Anime

Endpoint untuk mendapatkan gambar anime.

GET
/images/:category

Kategori yang tersedia:

  • naruto, aot, random
  • hd, 4k, landscape
{
  "status": "success",
  "data": {
    "url": "https://animeapi.site/images/naruto/1.jpg",
    "source": "Naruto Shippuden",
    "resolution": "1920x1080"
  }
}

Contoh Implementasi

Berikut contoh implementasi menggunakan JavaScript:

// Mendapatkan quote anime random
async function getRandomQuote() {
  try {
    const response = await fetch('https://animeapi.site/api/quotes');
    const data = await response.json();
    
    if (data.status === 'success') {
      const quote = data.data[0];
      console.log(`${quote.quote} - ${quote.character}`);
    }
  } catch (error) {
    console.error('Error fetching quote:', error);
  }
}

getRandomQuote();

WhatsApp Bot

Gunakan API untuk membuat bot WhatsApp yang bisa mengirimkan quotes anime, stiker, dan gambar secara otomatis.

// Contoh kode untuk bot WhatsApp
bot.onMessage((message) => {
  if (message.body === '!quote') {
    fetch('https://animeapi.site/api/quotes')
      .then(res => res.json())
      .then(data => {
        const quote = data.data[0];
        bot.sendMessage(message.from, `${quote.quote}\n\n- ${quote.character}`);
      });
  }
});

// Contoh kode JavaScript untuk website
async function loadTopCharacters() {
  const response = await fetch('https://animeapi.site/characters/popular?limit=5');
  const { data } = await response.json();
  
  const container = document.getElementById('characters-container');
  container.innerHTML = '';
  
  data.forEach(character => {
    const card = `
      

        
        
${character.name}

        
${character.anime}

    container.innerHTML += card;
  });
}

Discord Bot

Buat bot Discord yang bisa memberikan trivia anime dan fakta menarik.

// Contoh kode Discord.js
client.on('message', message => {
  if (message.content === '!trivia') {
    fetch('https://animeapi.site/trivia/random')
      .then(res => res.json())
      .then(data => {
        const trivia = data.data;
        const embed = new MessageEmbed()
          .setTitle('Trivia Anime')
          .setDescription(trivia.content)
          .setFooter(`Sumber: ${trivia.source}`);
        
        message.channel.send({ embeds: [embed] });
      });
  }
});
